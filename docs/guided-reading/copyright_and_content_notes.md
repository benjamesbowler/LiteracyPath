# Copyright and Content Notes

Source PDF: https://filecabinet9.eschoolview.com/D64E465C-E09E-4F7D-96F6-C12A63B1F126/ChildrensBookswithNoCopyright.pdf

The public-domain Guided Reading library is based on books listed in the provided no-copyright/public-domain source list. This implementation uses a curated subset for K-2 guided reading rather than importing the entire list.

## Deployment Caution

- Public-domain status should still be manually verified for the exact scan/edition before commercial deployment.
- Some old children's books contain outdated language, stereotypes, gender assumptions, religious framing, or frightening material.
- Books are selected for classroom suitability, but page-level review is still required after PDF download and image rendering.
- Books remain inactive until source PDFs, page images, covers, and manifests are validated.

## Content Standard

Included books should support guided reading, read aloud, fluency, vocabulary, sequencing, retell, character/setting work, prediction, and main idea/details. Dense, confusing, culturally outdated, violent, or overly archaic books should remain excluded or adapted only through a separate controlled writing process.
