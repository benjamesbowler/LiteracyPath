# Kimi Learn Area Image and Audio Request

## Reference Deck Source

Use `/Users/benjaminbowler/Desktop/EL_Skills_Block_All_81_PPTs.zip` as the layout/style reference.

The PPT pack has 81 decks: Cycle 1 through Cycle 27, three lessons per cycle. The reference decks use a clean classroom-slide system:

- 16:9 classroom slides
- large left-side letter or word focus
- clean right-side picture cards
- simple title-only divider slides
- warm-up game slides with a compact top bar and two or three answer cards
- no busy backgrounds
- no decorative clutter
- image placeholders such as `[Image: Mouse]`, not embedded copied images

Kimi assets should support that clean deck style: clear object/animal images, stable card crops, simple backgrounds, no text inside images, and no hotlinked external media.

## Global Image Requirements

- Format: `1024x1024 WebP`
- Style: natural-color kindergarten educational illustration, clean and simple
- Background: white, transparent, or very pale neutral
- Subject: one clear object/animal/action, centered, large silhouette
- Do not include text, labels, letters, numerals, watermarks, logos, brand packaging, sparkles, rainbow effects, or object faces unless the target is a living thing
- Keep crops consistent so the images work inside equal-height classroom slide cards

## Global Audio Requirements

- Format: `MP3`
- Voice: warm, clear adult female teacher voice, American English
- Delivery: calm, kindergarten-friendly, slightly slower than normal adult speech
- No music, sound effects, background noise, intro, outro, spelling unless requested, or extra words
- Normalize volume consistently
- Include a tiny silence at the beginning and end
- For word audio, say exactly the target word once
- For sound audio, say exactly the phoneme or teacher cue listed
- For chant audio, use only original LiteracyPath text from this document
