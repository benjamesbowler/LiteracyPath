# Kimi Learn Area Missing Visuals Request

The Learn Area currently uses CSS letter tiles, sound bubbles, word chips, and teacher-controlled video search links so the UI does not depend on missing images. These optional visuals would make the classroom lesson deck richer when generated.

Image style for all requests:
- normal natural-colored kindergarten cartoon
- simple object, clear silhouette, simple background
- no faces on objects
- no sparkles
- no rainbow colors unless the object is naturally rainbow-related
- no babyish styling
- educationally accurate
- transparent or simple light background preferred
