# Round 2 Level A-B Fiction Kimi Image Request

Date: 2026-05-25

## Scope

Create images only for 10 guided fiction books: 5 Level A and 5 Level B. Do not create audio from this request.

## Global Image Rules

- Render assets only. Do not rewrite page text.
- No embedded text, title text, captions, labels, watermarks, or speech bubbles.
- Same warm child-friendly illustration style across every page of a book.
- Fiction continuity is mandatory. If a character has the same shirt on page 1, they must have that shirt on every page.
- Keep skin tone, hair color/style, fur/feather colors, clothing, accessories, height/build, and age/species identical across pages.
- Keep environment continuity: same world, season, time-of-day logic, lighting, recurring props, and setting geography.
- Natural colors only. No rainbow-colored ordinary objects, sparkles, dark backgrounds, fantasy effects, or decorative clutter.
- Mobile-readable composition with the main action large and clear.
